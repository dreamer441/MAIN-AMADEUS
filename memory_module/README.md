# AMADEUS Memory Module

The Memory Module stores explicit AMADEUS memory that Dato marks through annotations.

This is not automatic memory and not hidden reasoning. V1 only saves memory when Dato uses:

```text
[memory][global] text
[memory][chat] text
```

Global memory travels across all chats. Chat memory belongs only to the active chat.

Memory lists are displayed in the right-side Memory panel so the main chat stays clean.
